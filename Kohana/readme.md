# BlogMVC with Kohana

A blog with the Kohana framework (PHP)

## Install

* Edit the bootstrap file:

    Kohana::init(array(
	'base_url'   => '/Kohana/'
	));

* Change '/Kohana/' with the folder where the index.php file is, for example '/blogmvc/Kohana/' if the index.php file is in /www/blogmvc/Kohana/.

* Import Dump

* Enjoy
